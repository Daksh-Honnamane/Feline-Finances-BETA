Feline Finances README

Thank you for checking out Feline Finances!

Feline Finances is a virtual pet simulation game where you care for your cat, manage its needs, and explore an interactive store to buy items. Your goal is to keep your cat happy, healthy, and well-fed while balancing finances.

How to Run

Currently, we're in the developer beta, so setup might be a bit more complex, but future versions will be easier to run!

  1. Make sure Python is installed
    You can download Python from here.
  2. Install Pygame
    Pygame is the framework we use to build the game. Install it by running the following command in your terminal: pip install pygame
  3. Run the Game
    To start the game, open the terminal in the folder containing main.py and run: python main.py

Controls

  • Arrow keys – Navigate through menus or make selections
  • Enter – Confirm choices
  • Esc – Quit the game
  • Mouse – Click buttons and interact with elements in the store or main gameplay screen

Features

  • Manage your cat's hunger, happiness, energy, cleanliness, and health
  • Store for purchasing items like food and toys
  • JSON-based save system to save progress
  • Interactive UI with buttons and dynamic feedback
  • Sound effects for actions like feeding, playing, and buying items
  • Animations and visual feedback based on your cat’s mood and status
